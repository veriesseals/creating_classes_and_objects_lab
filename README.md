# creating_classes_and_objects_lab

Java Car Collection Application

A simple Java console application demonstrating Object-Oriented Programming (OOP) concepts such as classes, objects,
arrays of objects, methods, and user input using Scanner.

------------------------------------------------------------

FEATURES
• Car class with common vehicle attributes
• Array-based storage for multiple cars
• Console-based user input
• Formatted output of car details
• Demonstrates core Java OOP principles

------------------------------------------------------------

PROJECT STRUCTURE

Main.java   - Application entry point
Car.java    - Car class definition
README.md   - Project documentation

------------------------------------------------------------

TECHNOLOGIES USED
• Java (JDK 8+)
• Java Scanner API
• Command-Line Interface

------------------------------------------------------------

CLASS OVERVIEW

Car.java
Fields:
- make (String)
- model (String)
- color (String)
- year (int)

Method:
getCarDetails(): returns a formatted String of car information.

Main.java
- Handles user input
- Stores multiple Car objects in an array
- Displays all stored cars

------------------------------------------------------------

HOW TO RUN

1. Compile:
javac Main.java Car.java

2. Run:
java Main

------------------------------------------------------------

LEARNING OBJECTIVES
• Understand object-oriented programming
• Create and use custom classes
• Store objects in arrays
• Use Scanner for user input
